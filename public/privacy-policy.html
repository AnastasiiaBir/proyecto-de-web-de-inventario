Política de Privacidad

Este sitio web es un proyecto educativo y no recoge datos personales de usuarios reales. Sin embargo, si se registraran datos en formularios de prueba, estos se tratarían de la siguiente manera:

1. Responsable del tratamiento

El responsable de los datos es el propietario del proyecto educativo, ubicado en Barcelona.

2. Datos personales recogidos

Se pueden recoger datos como nombre, correo electrónico y número de teléfono únicamente en formularios de prueba para fines de aprendizaje.

3. Finalidad del tratamiento

Los datos se utilizan únicamente para probar el funcionamiento de formularios y bases de datos en el proyecto educativo.

4. Conservación de los datos

Los datos se almacenan temporalmente en la base de datos local del proyecto y pueden eliminarse en cualquier momento.

5. Derechos de los usuarios

Los usuarios tienen derecho a acceder, rectificar o eliminar cualquier dato introducido durante las pruebas del proyecto.

6. Seguridad de los datos

Se aplican medidas de seguridad básicas para proteger la información almacenada en la base de datos local.

7. Contacto

Al tratarse de un proyecto educativo, no se proporciona contacto real.
